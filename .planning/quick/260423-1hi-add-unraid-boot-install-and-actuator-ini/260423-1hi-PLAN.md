# Quick Task 260423-1hi: Add Unraid boot install and actuator init scripts

## Goal

Add repo-managed shell scripts that bootstrap `unraid-actuator` on an Unraid host and provide a concrete initialization template for first-time setup.

## Tasks

1. **Add the boot-time install script**
   - Files: `scripts/1_actuator-install.sh`
   - Action: Install `uv`, refresh `PATH`, install Python 3.13 through `uv`, and install the `unraid-actuator` tool from GitHub in a way that can be rerun safely.
   - Verify: `bash -n scripts/1_actuator-install.sh`
   - Done when: The script is present, executable, and matches the requested Unraid boot flow.

2. **Add the initialization template and repo pointer**
   - Files: `scripts/4_actuator-init.sh.template`, `README.md`
   - Action: Add a template that copies EJSON keys, locks down permissions, runs init with placeholders, then validates and builds; document the scripts briefly in the README.
   - Verify: `bash -n scripts/4_actuator-init.sh.template`
   - Done when: The template is present with clearly replaceable placeholder values and the repo root points readers to the new scripts.
